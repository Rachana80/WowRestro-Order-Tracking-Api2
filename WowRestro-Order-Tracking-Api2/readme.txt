Uses WP Rest API to enable order related API endpoints.

== Description ==
WOWResto Order Tracking API is an addon to the popular food ordering platform for WOWRestro. Using this plugin Rest API endpoints can be activated for Order and payments related operations.

== Installation ==
1. Download the plugin and unzip it.
2. Upload/Copy the folder *wowrestro_order-api* to your /wp-content/plugins/ folder.
3. Activate the plugin from your WordPress admin panel.
4. Installation finished.

== Frequently Asked Questions ==

= Does this plugin work with multi site? =
Yes, this plugin is compatible with multi site.

= Does this plugin have any dependency? =
Yes, it needs the parent plugin WOWRestro to be installed and activated.
